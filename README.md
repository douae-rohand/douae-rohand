## About Me

<img align="right" width="45%" src="./assets/Gemini_Generated_Image_o2zmrso2zmrso2zm-removebg-preview.png" alt="My Avatar" />

I'm currently a **computer science engineering student** at the **National School of Applied Sciences of Tetouan**, who learns by building and breaking things until they work properly.

I’ve built real projects, including:
- **Academic projects** involving parsers, and system modeling
- **Web applications** (full-stack projects, APIs, and deployment)
- **Windows Forms desktop applications** with user interfaces and functional logic
- **Dockerized applications** designed for real-world deployment
- **Multiplayer and solo Java games** with clear separation between UI and game logic

I care about understanding how things work, not just making them run. If something is fragile, I refactor it. If it’s unclear, I redesign it.
